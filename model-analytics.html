Performance evaluation of the AI-based rice disease detection model

Model Overview

The AgriCare system uses a Convolutional Neural Network (CNN) trained on labeled rice leaf images to identify diseases.

Dataset Information

  • Dataset source: Public agricultural image dataset
  • Number of images: 3,000+
  • Classes: Multiple rice diseases
  • Image type: RGB leaf images

Model Accuracy

The trained model achieved an overall accuracy of:

92%

Evaluation Metrics

  • Accuracy: 92%
  • Precision: High
  • Recall: High
  • F1-Score: Balanced
